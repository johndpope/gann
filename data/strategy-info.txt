cut-45-006:
    x 採用基數轉換機制
    x 基本上以越過／跌破切線為進場依據
    x 需考慮當角度線發揮支撐／壓力時，該如何作單
        x 幾格之內算是有支撐／壓力？
            x 4 格（含）
            x 支撐／壓力發揮作用後，反向確立短線高點／低點後，即可進單
        x 突破幾格之內算是誤差？
            x 4 格（含）
            x 在誤差範圍之內，視為有支撐／壓力，反向確立短線高點／低點後，即可進單
    x 需考慮當盤勢在角度線來回振盪時要如何操作
        x 前期無需這麼複雜，一樣是以越過／跌破切線為進場依據，同一日一個方向只作一次單
    > 先前跌破角度線，之後就沒再跌破過
        x 屬進階研判，待研判更成熟時才加入
    x 如何停利？
        x 在切線角度有支撐／壓力
            x 切線角度前後 4 格（含）都算
            x 反向確立短線高點／低點後，即為停利時機
        x 高低點不在切線角度旁
            x 以跌破／越過切線角度為停利時機
    x 設 30 點停損
    x 不設保護傘
    x 比對時間從 2003-1-1 開始

cut-45-007:
    x 參考 cut-45-006 的操作條件
    x 避開盤整的方式
        1. 在切線角度上有表現出撐壓，卻來回振盪
            x 一開始不知是盤整，仍是以遇撐壓方式來操作
            x 發現有振盪現象，就得停止交易
            x 如未超過切線角度，則以越線作為進場條件（可立即下單）
            x 如已突破切線角度，但仍在誤差範圍內，則以越高／破低來恢復交易（但當次不作單）
        2. 越過切線角度一段距離，卻又回頭來測
            x 第一次回頭時，視為走勢反轉，越過切線角度時即可下單（可立即下單）
            x 如反轉後又來測相同角度，有可能是盤整，應暫時停止交易
            x 以越高／破低來恢復交易（但當次不作單）
    x 思考過程
        1. 在切線角度上來回大幅振盪
            x 發現有此現象時，暫時停止交易
            x 以越高／破低來恢復交易
        2. 突破切線幾格
            x 在誤差範圍內，視為有撐壓？
                x 視為有撐壓，按遇撐壓方式作單（第 3 項）
                x 如遇撐壓後，又再度來測，就視為第 1 項
            x 超過誤差後，視為第 1 項
        3. 在兩個切線角度間來回（未超過切線）
            x 一開始可能因在切線角度有撐壓而作到單
            x 發現無法獲利時，就該小心是在盤整
            x 以越過其中一個切線角度，才恢復交易

    - 停利：研判趨勢與持單方向相反時，即為停利時機
